#### アクション ツールバー

アクション バーは、画面の上部に表示されます。アクション バーの各種ツールを使用して、コンテンツの作成、編集、共有を行うことができます。![](images/action tool bar.PNG)

1. ギャラリー メニュー
2. 元に戻す/やり直し
3. 選択/計測
4. 作成/修正ツール
5. 位置/解析ツール
6. ユーザ ID
